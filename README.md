````markdown
# 🚀 Rádio Sinta — Premium Streaming Platform

Uma plataforma web moderna, imersiva e de alta performance desenvolvida especificamente para transmissão de streaming de áudio ao vivo 24/7. O projeto substitui interfaces legadas de web-rádios por uma arquitetura focada em Design Dopaminérgico, mobile-first absoluto, micro-interações fluidas e funis otimizados de engajamento social e conversão institucional.

---

## 💎 Diferenciais do Projeto & Engenharia de UI

- **Background Live Streaming (Sticky Player):** O player externo de áudio opera de forma ininterrupta no DOM através de um iframe desacoplado da renderização visual. Isso impede cortes ou reinicializações do streaming quando o painel de controle do usuário é expandido ou recolhido.
- **Fidelidade Estrita de Branding:** Componentização avançada de estilos com gradientes dinâmicos aplicados via CSS (`bg-clip-text`), mimetizando em texto puro de alta performance a identidade visual tridimensional e cromática do logotipo original.
- **Transições de Seção de Alto Nível:** Fusão imperceptível de ambientes utilizando múltiplos _color-stops_ posicionais em gradientes lineares. Isso elimina o choque térmico visual entre o Dark Mode Premium predominante e seções puristas claras (como o bloco de apoio emocional do CVV).
- **Grid Adaptável e Conversão em Três Pilares:** Arquitetura de informação baseada em cards no formato _Glassmorphism_ (Efeito Vidro/Desfoque com luz de fundo ambiente), dividida de forma simétrica entre curadoria musical, palavra devocional e integração social instantânea com o Instagram.

---

## 🛠️ Stack Tecnológica

- **Frontend Core:** React 18 / 19 + TypeScript (Tipagem estrita e segurança em tempo de compilação)
- **Build Tool & Bundler:** Vite (Ambiente otimizado com Fast Refresh)
- **Estilização:** Tailwind CSS (Arquitetura baseada em classes utilitárias modernas)
- **Animações:** Framer Motion (Orquestração de transições complexas baseadas em estados)
- **Icons:** Lucide React

## ⚙️ Instalação e Execução Local

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/radio-sinta.git](https://github.com/dborgesdev/radiosinta.git)
   cd radio-sinta
   ```
````

2. **Instale as dependências do projeto:**

```bash
npm install
# ou se preferir pnpm / yarn:
pnpm install

```

3. **Inicie o servidor de desenvolvimento local:**

```bash
npm run dev

```

4. **Build de Produção:**

```bash
npm run build

```
